## TC-005 — Граничное значение суммы с учётом комиссии

| Поле | Значение |
|------|----------|
| **Предусловие** | Сервис запущен, `balance=30000`, `reserved=20001` (доступно: 9999 ₽). Введён корректный номер карты `1111 1111 1111 1111`. |
| **Шаги** | 1. В поле суммы перевода ввести `9091`. |
| **Ожидаемый результат** | Правильная комиссия: `floor(9091 × 0.1) = 909 ₽`. Итого: `9091 + 909 = 10000 > 9999`. Отображается «Недостаточно средств на счете». |
| **Фактический результат** | Комиссия отображается как `900 ₽` (вместо `909 ₽`). Итого: `9991 ≤ 9999`. Кнопка «Перевести» отображается. |
| **Статус** | ❌ Не пройден |
| **Дефект** | [BUG-002](bug_report_002.md) |
